# Lil Dude

Your personal AI executive assistant. Secure, efficient, affordable.

## Quick Start

```bash
npm install
npm run build
npx lil-dude onboard
npx lil-dude start
```

## Development

```bash
npm run dev          # Start with hot reload
npm run test         # Run tests
npm run typecheck    # Type check
npm run lint         # Lint
```
